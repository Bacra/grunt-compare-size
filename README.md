# grunt-compare-size

Compare size of t

## Getting Started
Install this grunt plugin with: `npm install -g grunt-compare-size`

Then add this to your project's `grunt.js` gruntfile:

```javascript
task.loadNpmTasks('grunt-compare-size');
```
